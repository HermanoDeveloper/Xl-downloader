Bem-vindo(a)!

Veja como é fácil usar o XL Status Saver.

1. Navegue pelos Status

Na tela inicial, você pode ver todos os status recentes, filtrando por vídeos ou fotos.

2. Baixe e Compartilhe

Use os ícones em cada status para baixar o arquivo para o seu dispositivo ou compartilhá-lo com amigos.

3. Veja sua Galeria

Clique na aba "Salvos" no menu inferior para ver, compartilhar ou deletar todos os status que você guardou.

Atenção: Esta é uma versão de demonstração. O acesso aos status reais do WhatsApp requer uma integração nativa do aplicativo.
